
Capel Stock Verre - V1
======================

Instructions pour hébergement et lancement:

1. Installer Node.js sur le serveur
2. Copier les dossiers backend, frontend, database
3. Installer les dépendances:
   cd backend
   npm install
   cd ../frontend
   npm install
   npm run build
4. Lancer le serveur:
   cd backend
   node server.js
5. Accéder via navigateur:
   http://<adresse-de-ton-serveur>
6. Sur mobile: ajouter à l'écran d'accueil (PWA)
